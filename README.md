# Сайт услуг механической обработки

Презентационный одностраничный сайт для B2B: услуги, примеры работ с фото и описанием, блок «О компании», процесс работы, форма заявки.

## Запуск

Дважды кликните по `index.html` — сайт откроется в браузере. Сервер не нужен.

## Как добавить свои фото

1. Положите изображения в папку `assets/works/` (например `work-1.jpg`).
2. Откройте `data.js` и отредактируйте массив `WORKS`: заголовок, материал, операции, отрасль, путь к файлу.
3. Для главного баннера добавьте файл `assets/hero.jpg`.

## Бренд

Выбранный логотип: **вариант 5 «Шпиндель»** (`assets/logo.svg`). Запись в `brand.json`.

## Что настроить под себя

| Файл | Что менять |
|------|------------|
| `index.html` | Название «Вексель», телефон, email, адрес, цифры в блоке hero |
| `data.js` | Услуги (`SERVICES`) и портфолио (`WORKS`) |
| `main.js` | Email в `mailto:` для формы |

Форма сейчас открывает почтовый клиент (`mailto:`). Для отправки на сервер подключите Formspree, свой API или CRM.
